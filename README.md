# movietime

## Install
This project requires and API key from OMDB and the GCloud Natural Language API.

## Run
```
$ go build
$ ./movietime --lang no batman
Batman
⭐️⭐️⭐️⭐️⭐️⭐️⭐️
Gotham city. Kriminalitetssjef Carl Grissom (Jack Palance) driver byen effektivt, men det er en ny krimkjemper i byen - Batman (Michael Keaton). Grissoms høyre mann er Jack Napier (Jack Nicholson), en brutal mann som ikke er helt tilregnelig ... Etter å ha falt ut mellom de to Grissom har Napier satt opp med politiet og Napier faller til hans tilsynelatende død i en mengde kjemikalier . Imidlertid dukker han snart opp igjen som The Joker og starter en terrorperiode i Gotham City. I mellomtiden er reporter Vicki Vale (Kim Basinger) i byen for å gjøre en artikkel om Batman. Hun innleder snart et forhold til Batmans hverdagslige persona, milliardæren Bruce Wayne.
```
